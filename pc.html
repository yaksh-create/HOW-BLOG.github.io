HOW TO MAKE YOUR PC WELCOME YOU..



HOW DOES IT WORK?
The method takes the help of a text file to play the computerized audio recording of any welcome message you would like to hear at WINDOWS startup. Follow the steps to add your custom welcome message.

STEP 1:-
Open a Notepad file and type the folloing code:
dim speech welcome="Welcome my master, let's play with security" set speech=CreateObject("sapi.spvoice") speech.speak welcome
STEP 2:-
Now, save the notepad file as welcome.vbs. You can text the file if it works or not. Just double-click on the file and the sound will start playing. Else, error message will be displayed.
STEP 3:-
Now open the RUN dialog box by pressing WINDOWS + R. Now, type shell:startup and press enter.
STEP 4:-
Once you're in the Windows Startup Folder, copy the "welcome.vbs" file you created from desktop and paste it in the Startup folder.
Now restart your PC and listen to the magical voice.
NOTE:- You can change the text "Welcome my master, let's play with security" as per your liking. Make sure the inverted commas are present.
