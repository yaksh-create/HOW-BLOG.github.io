HOW TO INCREASE RAM OF YOUR PC..



YES, WE ARE GONNA INCREASE OUR RAM. NO, WE ARE NOT GONNA DOWNLOAD IT FROM INTERNET AND NO BS SOFTWARE.
HOW TO INCREASE?
THE CONCEPT:
So we are going to use the concept of Memory Paging to increase our RAM. Memory Paging is a memory managment technique for controlling how a computer's memory is shared.
WHAT DOES THAT MEAN?
We create a virtual memory, which is actually a section of our hard disk. We make this virtual memory emulate as computer's RAM. So when a computer runs out of RAM, the operating system will move pages of memory to this virtual memoy so that the actual RAM can work faster.
Let's create our own RAM..
STEP 1:-
Go to File Explorer and right click on "MY PC" or "MY COMPUTER" and click on Manage.
A computer management window will open with computer storage stats and lot of options.
STEP 2:-
Right click on "DISK MANAGEMENT" under Storage and click on Create VHD.
STEP 3:-
Now select a path to store that Virtual Memory/RAM and choose the size of that RAM. Don't choose more than 2GB. Keep the rest options as default and click on create. A disk will be created which act as RAM in your PC. Now time to intialize it.
STEP 4:-
Now right click on the newly created disk and choose "Initialize Disk" and then MBR and finally OK.
OUR VIRTUAL DISK IS READY TO WORK AS RAM..
